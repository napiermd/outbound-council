# GTM Council Roster

This council is a source-backed bench of real GTM, medtech, healthcare, and enterprise-sales voices.

The goal is not to imitate celebrity style. The goal is to distill each person's public operating logic into a useful decision lens for IntuBlade: single-use video laryngoscopy today, software and computer-vision procedural guidance tomorrow.

## Activation rules

- Do not activate everyone by default.
- Pick 2-4 voices for most questions.
- Use 5-6 voices only for major positioning, fundraising, or channel strategy decisions.
- If the question touches hospital workflow, patient safety, procedural guidance, or clinical AI, include Bob Wachter or Josh Makower.
- If the question touches airway technique, training, EMS protocol, or clinical safety, include at least one clinical airway voice.
- If the question must sound buyer-native, include at least one buyer archetype.
- If the question touches procurement, value analysis, or hospital committees, include Mark Copeland.
- If the question touches economics, reimbursement, budget impact, or value dossiers, load `playbooks/market-access-health-economics.md`.
- If a competitor or substitute is mentioned, load the relevant `competitive/` profile.
- If the question is a live deal, start with John McMahon and Matt Dixon.
- If the question is "why would anyone change?", start with Bob Moesta.
- If the question is "what category are we creating?", start with April Dunford and Christopher Lochhead.
