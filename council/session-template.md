# Session Template

Use this shape when invoking the GTM council.

```text
Situation: ...
Decision needed: ...
Buyer / market context: ...
Active voices: ...
Core tension: ...

Independent read:
- Voice: ...
  What they notice: ...
  Profile basis: [which profile sections matter: signature questions, IntuBlade applications, source confidence]
  What they would challenge: ...
  Recommended move: ...

Council meeting:
- Where the voices agree: ...
- Where they disagree: ...
- What the disagreement changes: ...

Recommendation:
- ...

Next move:
- ...
```

## Enriched voice mode

Use this when a profile has `IntuBlade applications`, `Source mining backlog`, and `Voice extraction targets`.

```text
Situation: ...
Pack: ...
Active voices: ...
Loaded profile sections:
- Voice: canonical sources / current signals / IntuBlade applications / extraction targets

Voice-specific application:
- Voice:
  IntuBlade translation: ...
  Likely objection: ...
  What this voice would rewrite or inspect: ...

Council synthesis:
- ...

Decision:
- ...

Next source to mine:
- ...
```

## Rating mode

Use for reviewing email drafts, campaign ideas, decks, sequences, offers, and deal plans.

```text
Situation: ...
Active voices: ...
Scoring criteria: ...

Independent scorecard:
- Voice: ...
  Rating: <x/10>
  Why: ...
  Keep: ...
  Fix: ...

Council synthesis:
- ...

Recommendation:
- ship / revise / do not send yet

Next move:
- ...
```

## Pack scorecard mode

Use when reviewing an outbound email, a campaign, a deck, a pilot plan, a value-analysis package, or a deal.

```text
Situation: ...
Pack: ...
Artifact reviewed: email / campaign / deck / deal / pilot / value-analysis package

Scores:
- Positioning clarity: <x/10>
- Buyer change trigger: <x/10>
- Trust and evidence: <x/10>
- Decision-risk reduction: <x/10>
- Next-step strength: <x/10>

Voice notes:
- Voice:
  Score: <x/10>
  Keep: ...
  Fix: ...
  Reason: ...

Blocking issues:
- ...

Revision:
...

Ship decision:
- ship / revise / do not send yet
```
