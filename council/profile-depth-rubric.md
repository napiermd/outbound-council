# Profile Depth Rubric

Use this to score whether a profile is ready for different kinds of work.

## Score 1: Nameplate

- Basic domain only
- Few or no sources
- No usable decision behavior

Use for: candidate tracking only.

## Score 2: Summary

- Canonical sources listed
- Basic worldview and questions captured
- Minimal source mining

Use for: light brainstorming.

## Score 3: Operational

- Strong canonical sources
- Current signals
- Signature questions and decision style are usable
- IntuBlade applications documented
- Source mining backlog exists

Use for: normal GTM council work.

## Score 4: High fidelity

- Multiple source extractions exist
- Language/story/framework banks are populated
- Tensions with other voices are specific
- Can review drafts and deal plans in a recognizable, non-generic way

Use for: campaign reviews, strategic narrative, deal inspection.

## Score 5: Live

- Refreshed in the last 90 days
- Recent-signal file exists
- Source extractions include current public output
- Profile is safe to use for high-stakes current claims

Use for: board/investor materials, public strategy, high-stakes positioning.
