# Competitive Intelligence Index

Use these profiles when drafting competitor-aware outbound, hospital value-analysis responses, positioning, and sales enablement.

## Profiles

| Profile | Type | Use when... |
| --- | --- | --- |
| Ambu | Portfolio competitor | Buyer values single-use airway visualization, bronchoscopy, or one-platform procurement. |
| Verathon GlideScope | Premium incumbent | Buyer values broad portfolio, clinical proof, first-pass success, and market leadership. |
| Medtronic McGRATH | Portable VL competitor | Buyer values handheld portability, familiar Macintosh technique, and large-company support. |
| Karl Storz C-MAC | Premium modular competitor | Buyer values modular reusable/single-use airway ecosystem and premium engineering. |
| VividTrac | Disposable/USB substitute | Buyer values low-cost disposable video intubation and open display options. |
| Direct laryngoscopy | Procedure substitute | Buyer believes direct laryngoscopy is adequate, cheaper, or important for skill retention. |
| Supraglottic-first strategies | Algorithm substitute | EMS or hospital teams prioritize SGA/BVM pathways over ETI/VL expansion. |
| Reusable capital-equipment workflows | Operating-model substitute | Buyer already owns reusable VL and is optimizing carts, reprocessing, and capital use. |

## Use Rules

- Treat competitor claims as claims unless verified independently.
- Never disparage a competitor's safety or quality without direct evidence.
- Counter-position around IntuBlade's likely wedge: disposable video airway coverage, readiness, deployment, training, reviewability, and low-friction access.
- Pair with `playbooks/market-access-health-economics.md` for economic claims.
