# Buyer Archetype Index

Use these profiles as buyer-truth lenses, not as celebrity voices.

Each archetype should be activated alongside the expert council when the artifact must sound like the buyer's world: outbound, call prep, value-analysis packets, pilot design, objection handling, and win/loss review.

## Use Rules

- Pair at least one buyer archetype with every outbound, hospital deal, or pilot review.
- Separate direct buyer language from council inference.
- Upgrade `Evidence status` to `field-note backed`, `interview-backed`, or `live` only when the repo contains sanitized field signals.
- Never include PHI or private identifiers unless intentionally approved and scrubbed.
